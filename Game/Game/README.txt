# Lion King

------------------------
HOW TO COMPILE AND RUN
------------------------
Compile: g++ -std=c++17 Board.cpp Player.cpp Game.cpp Tile.cpp Driver.cpp -o proj2
Run: ./proj2.exe
------------------------
DEPENDENCIES
------------------------
Board.h, Game.h, Player.h, Tile.h, characters.txt, random_events.txt must be in the same
directory as the cpp files in order to compile.

ABOUT THIS PROJECT
------------------------
This project implements an interactive board game fashioned after the game
Game of Life (Base off Lion King). Up to 5 players can play the game. You navigate your chosen Path
which is filled with various events that happen along the way! The winner is decided by who has the most Pride Points! Good luck!
